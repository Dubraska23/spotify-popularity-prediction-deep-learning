# spotify-popularity-prediction-deep-learning
Predicción de popularidad de canciones en Spotify mediante Redes Neuronales Profundas (Keras/TensorFlow). Implementa arquitecturas densas con Dropout, BatchNormalization y optimización avanzada (Adam). Enfoque en Model Governance, ética en algoritmos de recomendación y gestión de activos de IA para Business Analytics (UAM).
